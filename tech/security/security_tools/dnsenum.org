* dnsenum
Github: [[https://github.com/fwaeytens/dnsenum][dnsenum]]
* Description
Pulls dns records of specified domain

* Usage
> dnsenum <domain>

** Options
w - whois queries
o - specifies output location
d - sets the delay between whois lookups
r - recursive lookups
