*Changing a users shell*

[[evernote:///view/47830806/s238/6795193b-0a9f-4698-89ca-e95faf33c124/6795193b-0a9f-4698-89ca-e95faf33c124/][chsh]]

Change a users shell

> chsh -s /bin/shell -u <login>

*Adding a group*

[[evernote:///view/47830806/s238/2b7470d8-bdb6-462b-86c2-eaaf6069f427/2b7470d8-bdb6-462b-86c2-eaaf6069f427/][groupadd]]

> groupadd <group>

*Deleting a group*

[[evernote:///view/47830806/s238/2b7470d8-bdb6-462b-86c2-eaaf6069f427/2b7470d8-bdb6-462b-86c2-eaaf6069f427/][groupdel]]

> groupdel <group>

*usermod*

Adding an existing user to a group

> usermod -G <group> <user>

*Useful commands for user information*

What groups is a user in?

> cat /etc/group | grep <user>
