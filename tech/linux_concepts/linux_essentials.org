- Setting up VNC Server and Connecting with VNC client
  - check it is installed
    #+BEGIN_SRC sh
    which vncserver
    #+END_SRC
    this should return the vncserver path
  - set the vnc password
    #+BEGIN_SRC sh
    vncpasswd
    #+END_SRC
  - start the vncserver
    #+BEGIN_SRC sh
    vncserver
    #+END_SRC
  - vnc configuration files are kept in ~/.vnc
  - kill the vncserver
    #+BEGIN_SRC sh
    vncserver -kill :1
    #+END_SRC
    :1 is the instance number(5901)
