* uniq
* Description
* Usage
- print only unique lines in a file
#+BEGIN_SRC sh
uniq text.txt
#+END_SRC

- print the duplicate lines
#+BEGIN_SRC sh
uniq -d text.txt
#+END_SRC

- print all instances of duplicate lines
#+BEGIN_SRC sh
uniq -D text.txt
#+END_SRC
