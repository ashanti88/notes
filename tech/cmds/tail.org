* tail
* Description
* Usage
> tail <option> <file>

- print a number of lines
#+BEGIN_SRC sh
tail -n 5 file.txt
#+END_SRC

- follow a file with tail
#+BEGIN_SRC sh
tail -f file.log
#+END_SRC
this command is helpful to follow a log file as it outputs/prints information
