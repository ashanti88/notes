#+TAGS: sec av


* clamscan							     :sec:av:
* Description
* Usage
> clamscan <file>

** Scan a directory
#+BEGIN_SRC sh
clamscan -r /home/crito/directory
#+END_SRC

** Print only the infected files
#+BEGIN_SRC sh
clamscan -r -i /home/crito/directory
#+END_SRC

** Print the infected filenames to a file
#+BEGIN_SRC sh
clamscan -r /home/crito/directory | grep FOUND >> report.txt
#+END_SRC

** Updating clamav
#+BEGIN_SRC sh
freshclam
#+END_SRC
