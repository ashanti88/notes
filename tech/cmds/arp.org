#+TAGS: net op


* Arp
* Description
This command is used to dump the kernel arp cache (/proc/net/arp).

* Usage
> arp <options>

** Options
a - all entries, dump all of the cache
s - set a new entry
d - delete an entry
i - specify a sepecific interface
