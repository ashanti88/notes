#+TAGS: pgrep process


* pgrep
* Description
This cmds will print all associated pids that a command is currently running

* Usage
- find the pid of the bash process
#+BEGIN_SRC sh
pgrep bash
#+END_SRC

- find the pid of the init process
#+BEGIN_SRC sh
pgrep init
#+END_SRC
