#+TAGS: sys op file


* find								:sys:op:file:
* Description
Used for finding files by file name. 
* Usage

> find <start_point> <options> <pattern>

** Delete found files
#+BEGIN_SRC sh
find . -name "pattern" -exec rm -rf {} \;
#+END_SRC
** Count files in directory recursively
#+BEGIN_SRC sh
find . -type f | wc -l
#+END_SRC
