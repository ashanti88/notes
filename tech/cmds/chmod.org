* chmod
* Description
Command to change the file mode bits.

* Usage

> chmod XXX file.txt

- Options
u - user
g - group
r - read
w - write
x - execute

** Make a file executable
#+BEGIN_SRC sh
chmod +x file.py
#+END_SRC
