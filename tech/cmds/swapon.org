#+TAGS: memory swap swapon


* swapon
* Description
This command is used to initialise a swap file

* Usage
- check if any swap has been activated
#+BEGIN_SRC sh
swapon -s
#+END_SRC

- create a swap file
#+BEGIN_SRC sh
dd if=/dev/zero of=/swapfile bs=1G count=4
chmod 600 /swapfile
mkswap /swapfile
swapon /swapfile
#+END_SRC
