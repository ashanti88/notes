


* iptables						     :sys:net:sec:fw:
* Description
* Usage
** List current rules
> iptables -L
this may take a while as it will attempt to resolve all the ip addresses
> iptables -nL
this will disable name resolve

** Flush currnt rule set
> iptables -F
or
> iptables --flush
