* zipinfo							  :file:anal:
* Description
this cmd can list detailed information about an archive
* Usage
> zipinfo <option> <filename.zip>

** Options
1 - list filenames only
2 - list filenames only but include headers
s - list zipfile info in short Unix "ls -l" format
m - similar to above
l - list in long form
v - list in verbose
