* mkfs
* Description
* Usage
> mkfs -t <fs> <device>
t - filesystem type

- create a filesystem on partition
#+BEGIN_SRC sh
mkfs -t ext2 /dev/sdb1
mkfs -t ext3 /dev/sdb2
mkfs -t ext4 /dev/sdb3
#+END_SRC

* mkfs.ext
** Create filesystem on device
#+BEGIN_SRC sh
mkfs.ext2 /dev/sda1 
#+END_SRC
