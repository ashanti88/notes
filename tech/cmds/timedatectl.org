* Usage
- list all timezones
#+BEGIN_SRC sh
timedatectl list-timezones
#+END_SRC

- set the timezone
#+BEGIN_SRC sh
timedatectl set-timezone Europe/London
#+END_SRC

- show the current status of your data-time settings
#+BEGIN_SRC sh
timedatectl status
#+END_SRC
