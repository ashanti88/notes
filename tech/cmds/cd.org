#+TAGS: sys op


* cd								     :sys:op:
* Description
Change Directory
* Usage

> cd <options> <path>

** move to users home directory
#+BEGIN_SRC sh
cd
#+END_SRC
or
#+BEGIN_SRC sh
cd ~
#+END_SRC

** move to previous directory
#+BEGIN_SRC sh
cd -
#+END_SRC

** move back one directory
#+BEGIN_SRC sh
cd ..
#+END_SRC
