#+TAGS: su root switch_user


* su
* Description
Switch to another system user or root

* Usage
- switch to root
#+BEGIN_SRC sh
su -
#+END_SRC
the dash indicates to use the users .profile, .bash_profile if available

- switch to frank
#+BEGIN_SRC sh
su - frank
#+END_SRC
